Processus de signalement citoyen

Pour signaler un problème, les citoyens accèdent au site web et choisissent le quartier et le type de problème à signaler.

Dans le centre d’alerte, la Commune et les OSC reçoivent une alerte les informant du problème signalé.

Les OSC impliquées reçoivent une alerte pour se rendre sur place et résoudre le problème.

Les agents responsables se déplacent sur le site et résolvent le problème signalé.

Une fois le travail terminé, le site est mis à jour pour informer les utilisateurs que le problème a été résolu.
